# Kit de Facas - Landing Page

Pagina estatica pronta para publicar na Vercel.

## Publicacao

Use esta pasta como raiz do projeto na Vercel:

`E:\Ofertas-Code\landing-pages\kit-facas-v2`

Como existe um `index.html` na raiz, a Vercel publica a pagina diretamente. O arquivo `vercel.json` ativa URLs limpas.

Nome sugerido para o projeto na Vercel:

`kit-facas-pague-na-entrega`

Rota harmonica configurada:

`/kit-facas-pague-na-entrega`

Exemplo quando publicar:

`https://kit-facas-pague-na-entrega.vercel.app/kit-facas-pague-na-entrega`

Se preferir uma URL mais curta, use apenas a raiz do projeto:

`https://kit-facas-pague-na-entrega.vercel.app`
